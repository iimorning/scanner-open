# 贡献指南

感谢您对 Privacy Scanner 项目的关注！我们欢迎所有形式的贡献。

## 如何贡献

### 报告问题

如果发现 bug 或有功能建议，请在 GitHub Issues 中创建 issue。

### 提交代码

1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

### 代码规范

- 遵循 ESLint 规则
- 编写测试覆盖新功能
- 更新相关文档

### 测试

在提交 PR 前，请确保：
- 所有测试通过 (`npm test`)
- 代码通过 lint 检查 (`npm run lint`)
- 示例项目可以正常运行

## 行为准则

请保持友好和尊重，创建积极的协作环境。
