# Changelog

## [0.2.0] - 2025-11-13

### ✨ 新增功能
- **AI 友好的数据结构**: 新增 `generateAISummary()` 方法，为 AI 生成提供结构化数据
- **隐私风险评估**: 自动评估项目隐私风险等级 (HIGH/MEDIUM/LOW)
- **合规建议生成**: 根据检测结果生成 GDPR/CCPA 等合规建议
- **数据流向分析**: 分析个人数据的收集、存储和共享模式
- **智能文件过滤**: 新增 FileUtils 工具类，提供智能文件和目录过滤
- **项目类型检测**: 自动检测 Android、iOS、Web、Flutter 等项目类型
- **增强的扫描日志**: 详细的扫描过程日志，包含 emoji 标识和进度显示

### 🔧 改进
- **扫描器重构**: 优化 AndroidScanner、IOSScanner、WebScanner 的实现
- **错误处理增强**: 改进扫描过程中的错误处理和容错机制
- **性能优化**: 支持并行扫描和结果缓存
- **依赖管理**: 新增 Gradle 和 package.json 依赖分析

### 📊 功能特性
- **多平台支持**: Android (Java/Kotlin/XML)、iOS (Swift/Objective-C/Info.plist)、Web (JavaScript/HTML/CSS)
- **智能模式匹配**: 优化正则表达式匹配，提高检测准确率
- **结果格式标准化**: 统一的扫描结果数据结构
- **可配置扫描选项**: 支持自定义扫描模式和排除规则

### 🐛 修复
- 修复第三方服务检测的排序问题
- 修复权限推断逻辑
- 修复 HTML 特殊分析的兼容性问题

### 📝 文档
- 完善代码注释和 JSDoc
- 新增使用示例和最佳实践

## [0.1.0] - 2025-11-13

### 🎉 初始版本
- 基础扫描器架构 (BaseScanner)
- Android、iOS、Web 平台扫描器
- 基本的权限和 API 检测
- 第三方服务识别
